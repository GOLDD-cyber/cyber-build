# &lt;NEON_FORGE/&gt; — сервис сборки игровых ПК

Лендинг сервиса по сборке ПК в тёмно-зелёном киберпанк-стиле.

## Возможности
- Анимированный фон на главной (matrix «цифровой дождь» на `<canvas>`)
- Плавное появление блоков при прокрутке
- 3 готовые сборки на выбор
- Каталог отдельных компонентов (видеокарты, процессоры и др.)
- Отзывы клиентов
- Форма бесплатной консультации
- Кнопка «наверх», адаптация под телефоны

## Технологии
- **HTML** — структура
- **CSS** — киберпанк-оформление, неоновое свечение, анимации
- **JavaScript** — matrix-эффект, анимации прокрутки, интерактив

## Запуск
Открыть `index.html` в браузере.
